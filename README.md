## ✨ Key Features

### 🏠 Dashboard
Prominent introductory overview with KPI cards (Total Income, Total Expenses, Net Balance, Savings Rate), a donut chart of spending by category, a monthly bar chart, and recent transaction feed.

### 📊 Purpose & Skills
Demonstrates proficiency in vanilla HTML/CSS/JS, SVG chart rendering, DOM manipulation, Intl API (INR currency formatting), and responsive glassmorphism UI design.

### ⚙️ Workflow & Benefits
- Record income and expenses across categorised types
- Visualise spending patterns through live-rendered SVG charts
- Compare month-over-month performance in the Analytics tab
- Full add / delete / filter / search on all transactions

### 📬 Contact & Links
All source code is open on GitHub. The live site is deployed via GitHub Pages and accessible instantly — no login, no setup required.

## 🛠 Technical Stack

| Layer | Technology |
|---|---|
| Markup | HTML5 (semantic) |
| Styling | CSS3 — CSS variables, glassmorphism, backdrop-filter, keyframe animations |
| Logic | Vanilla JavaScript ES6+ — Intl API, SVG DOM, event delegation |
| Charts | Pure SVG (donut) + CSS flex (bar charts) — zero chart libraries |
| Fonts | Google Fonts — Syne (display) + DM Sans (body) |
| Deploy | GitHub Pages |

## 🎨 Design Identity

Crafted a bespoke **neo-purple glassmorphism** aesthetic inspired by modern fintech dashboards:
- Deep `#060609` background with dual radial purple ambient glows
- Glass cards with `backdrop-filter: blur(24px)` and inner glow borders
- Syne display font for headers, DM Sans for body copy
- Smooth page-transition animations and hover micro-interactions
- Fully responsive — mobile, tablet, and desktop

## 🚀 Running Locally

No installation needed. Just open `index.html` in any browser:

```bash
git clone https://github.com/Shlok1107141/nexvault.git
cd nexvault
open index.html   # or double-click the file
```
